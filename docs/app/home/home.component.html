Check Me Out!

If you can do it with CSS then you can do it with CBJ Scrollbar.

Fully Customizable

Since the directive prevents the browser from rendering a scrollbar and instead draws one, we are able to fully control the look through css properties. Properties can be set either through a convenient configuration object, or added via some css classes.

Angular Package Format

This package has been built with ng-packagr and meets the APF specs. It has been successfully tested with AOT compilation and can integrate easily into a project.

Where am I ?!

Auto hide the scrollbar and control the visibility timeout, all through a simple to use config class.
