Considerations & Tips

  • Directive must be placed on an element with a set height
  • The parent element will have its position set to relative
  • Consider setting the wrapper, grid, and bar styles in the config
  • Styles set in the config override all other styles, including those set internally
  • Depending on your layout it may be necessary to set a parent or grandparent container to overflow hidden to prevent browser scrollbar from appearing

Classes

  • cbj-scroll-wrapper
    : The scroll wrapper is an element created by the directive that wraps the element containing the directive and the created scrollbar and grid. 
  • cbj-scroll-grid
    • : The scroll grid is the container of the scrollbar. It wil stretch from the top of the scroll wrapper to the bottom. 
  • cbj-scroll-bar
    : The scrollbar itself.
